TAGS PARA LISTAS

Para poder criar uma lista, podemos utilizar uma lista ordenada, a partir das tags 'ol' e '/ol', ou uma lista não ordenada, a partir das tags 'ul' e '/ul'. Posteriormente, incluímos dentro da lista os elementos da mesma, dentro das tags 'li' e '/li'.
